JavaScript

A tale about an omnipresent language.
JavaScript is originally found in the browsers, but also
  • In browser extensions: like Chrome
  • Server side: Node
  • In the database: MongoDB
  • In the command line: through NPM packages
  • In music production tools: like Max/MSP
  • In desktop client, thanks to Node Webkit: like Popcorn Time
  • In microcontrollers: Tessel
JavaScript is:
  • Event driven
  1. Script is loaded
  2. Any execution can bind new listeners on events
  3. The event loop:
    1. Takes an incoming event
    2. Fires the relevant listeners if any
An event:
  • A target
  • A type
  • Attached data

Callbacks / closures

A function that we manipulate like a variable.

                function handleClick() {
                    console.log('My button has been clicked');
                }

                function bindListener() {
                    $('#myButton').bind('click', handleClick);
                }

                $(document).ready(bindListener);

Variables have a scope.

$(document).ready(function() {
    var i = 0;

    $('#myButton').bind('click', function() {
        console.log('My button has been clicked ' + i +
            ' times before.');
        i += 1
    })
});
